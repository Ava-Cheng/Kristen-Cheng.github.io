利用Hexo架設部落格

發表於 2020-02-05更新於 2020-02-05

前言

一開始想要建立部落格時，就會先遇到該選擇何種平台撰寫的困境，現今已有許多平台可供我們做選擇，例如Medium、Blogger、 痞客邦Pixnet等知名平台，用已經現有的平台有許多的優點，例如文章搜尋、有相對應的友善操作介面、該平台知名度夠大等等優勢，但相對也有版型限制、有些需要付費、文章發佈數量種種限制。最終考量客製化網頁的彈性，以及使用Markdown語法撰寫文章、不受平台本身限制所拘束總總考量，本次利用在GitHub Page來架設 Hexo 部落格，以下為架設記錄。

環境配置

  • 安裝Nodejs
  • 安裝Git

    $ npm install -g hexo-cli

更換主題

初始化設定完，預設主題為landscape，然而覺得太平凡想更換主題。(本次以stun作為範例)

  1. 前往挑選預想更換的主題>>Hexo主題.
  2. 安裝主題stun，並搬移到根目錄下的themes目錄下
    $ git clone https://github.com/liuyib/hexo-theme-stun.git stun
  3. 安装依賴 pug $ npm install --save hexo-render-pug
  4. 修改根目錄下的 _config.yml 文件。theme: stun # 注意！ themes目錄下主題名稱要跟此相同
  5. 重新啟動(hexo clean && hexo g && hexo s)

在GitHub上部署自己的網頁

最後當你建立好blog後，要放上去網路上讓大家都能瀏覽觀看，這次將使用Github Page作為範例。

  1. 在GitHub 創建一個名稱為username.github.io 的專案 # 注意！username必須要跟自己GitHub名稱一模一樣
  2. 安装 deploy git 插件$ npm install hexo-deployer-git --save
  3. 在主題配置文件 _config.yml 中修改deploy區塊

4.在主題配置文件 _config.yml 中修改urlurl: https://<username>.github.io/
5.最後部署到Github上即可( hexo clean && hexo g && hexo d)
